# Pi Agents

AI-driven planning and coding workflows for software projects using [Pi](https://pi.dev).

## Install

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/iapicca/pi_agents/HEAD/install.sh)"
```

## Usage

- **Plan your project:** `/plan "Add OAuth authentication with GitHub"`
- **Build your project:** `/code https://github.com/owner/repo/issues/42`

That's it — the extension handles the rest.
